TCP/IP

TCP/IP é um conjunto de protocolos que possibilita a comunicação entre computadores e servidores, formando a Internet. Ele combina o Protocolo de Controle de Transmissão (TCP), responsável pela transmissão de dados, com o Protocolo de Internet (IP), que identificar os computadores e servidores.

Camada 1 - Enlace

Esta camada corresponde principalmente à camada de enlace do modelo OSI. Ela lida com a transmissão de dados na forma de quadros de rede entre dispositivos na mesma rede local (LAN). Ela também trata do acesso ao meio e da detecção de erros. Protocolos comuns nesta camada incluem Ethernet e Wi-Fi.

Camada 2 - Rede

Corresponde à camada de rede do modelo OSI. Ela é responsável pelo roteamento de pacotes de dados através de redes. O protocolo IP (Internet Protocol) é o mais conhecido nessa camada e é usado para rotear dados de um nó para outro através da Internet. Também inclui o ICMP (Internet Control Message Protocol) para gerenciamento de mensagens e o ARP (Address Resolution Protocol) para mapear endereços IP em endereços MAC.

Camada 3 - Transporte

Equivale à camada de transporte no modelo OSI. A camada de transporte lida com a transmissão de dados de uma extremidade a outra e garante que eles cheguem sem erros. Os protocolos mais notáveis nesta camada são o TCP (Transmission Control Protocol), que fornece comunicação confiável, e o UDP (User Datagram Protocol), que fornece comunicação não confiável, mas mais rápida.

Camada 4 - Aplicação

Na camada de aplicação, ocorrem os processos e aplicativos que usam a rede para se comunicar. Ela é uma combinação das camadas de sessão, apresentação e aplicação do modelo OSI. Nesta camada, você encontrará uma ampla variedade de protocolos, incluindo HTTP (Hypertext Transfer Protocol) para navegação na web, SMTP (Simple Mail Transfer Protocol) para e-mail, FTP (File Transfer Protocol) para transferência de arquivos e muitos outros.

Camada 5 - Inter-rede

Essa é uma camada adicional que não é uma parte estrita do modelo OSI, mas é frequentemente mencionada no contexto do TCP/IP. Ela está relacionada à conectividade da Internet e ao roteamento de pacotes em escala global. O protocolo BGP (Border Gateway Protocol) é um exemplo de protocolo usado nesta camada para roteamento na Internet.
